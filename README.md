# KICB Test App - Справочник пользователей

Тестовое задание: Веб-приложение на ASP.NET Core для управления списком пользователей и их телефонных номеров.

## Функционал:
* **Управление пользователями:** Создание, просмотр, редактирование и удаление.
* **Телефонный справочник:** Привязка нескольких номеров к одному пользователю (связь 1:N).
* **Валидация:** Проверка обязательных полей и уникальности Email и номеров телефонов.
* **Интерфейс:** Построен на шаблоне MVC с использованием Bootstrap.

## Технологический стек:
* **Платформа:** .NET 10 / ASP.NET Core MVC
* **База данных:** MS SQL Server (LocalDB)
* **ORM:** Entity Framework Core (Code First)

## Инструкция по запуску:
1.  **Клонируйте репозиторий** на свой компьютер.
2.  Откройте решение (`.sln`) в **Visual Studio 2022**.
3.  Откройте **Консоль диспетчера пакетов** (Package Manager Console).
4.  Выполните команду для создания базы данных:
    ```powershell
    Update-Database
    ```
5.  Нажмите **F5** (или зеленую стрелку запуска), чтобы запустить приложение в браузере.

## Особенности реализации:
* Реализованы уникальные индексы в БД для предотвращения дубликатов.
* Настроены навигационные свойства для удобного отображения количества телефонов в списке пользователей.
